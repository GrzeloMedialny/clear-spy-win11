🛡️ Przewodnik: Jak oczyścić Windows 11 z funkcji szpiegujących i używać go offline pod Adobe

Cel: Stworzenie stabilnego, offline’owego systemu Windows 11 bez telemetrii i zbędnych usług, gotowego pod pracę graficzną (np. Adobe Photoshop, Illustrator).

1. Instalacja Windows 11 BEZ internetu

  • Odłącz kabel Ethernet lub wyłącz Wi-Fi podczas instalacji
  • Kiedy system żąda konta Microsoft, użyj kombinacji Shift + F10
  • Wpisz w oknie terminala:
    OOBE\BYPASSNRO
  • System zrestartuje się i umożliwi utworzenie lokalnego konta

2. Wyłącz i usuń usługi szpiegujące

Usługi do wyłączenia (naciśnij Win + R, wpisz services.msc):

  • Connected User Experiences and Telemetry
  • Diagnostics Tracking Service
  • dmwappushsvc
  • Windows Error Reporting Service
  • SysMain (Superfetch)
  • Windows Update (jeśli system ma pozostać offline)

Rejestr (Regedit)

Przejdź do klucza: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection

Dodaj wartości DWORD (32-bit):

  • AllowTelemetry = 0
  • DisableTelemetry = 1

Uwaga: Błąd w rejestrze może unieruchomić system. Wykonaj kopię zapasową!

3. Zablokuj połączenia wychodzące (Firewall)

  • Otwórz Zaporę systemu Windows i ustaw domyślnie blokowanie wszystkich połączeń wychodzących
  • Opcjonalnie: dodaj wyjątek dla Adobe Creative Cloud tylko na czas aktywacji

4. Narzędzia do automatycznego czyszczenia systemu

  • ✅ O&O ShutUp10++ — najlepszy, darmowy, bezinstalacyjny
  • WPD (Windows Privacy Dashboard)
  • Privacy.sexy lub Debotnet

5. Wyłączenie Windows Update (na zawsze)

  • Wyłącz usługę Windows Update w services.msc
  • Zablokuj adresy update.microsoft.com w pliku C:\Windows\System32\drivers\etc\hosts

6. Odcięcie systemu od Internetu

  • Usuń sterowniki karty sieciowej
  • Zablokuj połączenia przez ustawienia Firewalla lub BIOS
  • W razie potrzeby – fizycznie usuń kartę Wi-Fi lub wyłącz z poziomu UEFI/BIOS

7. Adobe w trybie offline – jak to zrobić?

  • Starsze wersje (np. CS6) działają w pełni offline
  • Creative Cloud — aktywuj raz z Internetem, potem blokuj połączenia (firewall/hosts)
  • Nie korzystaj z Adobe Fonts ani Cloud, jeśli chcesz pełną izolację

Alternatywa? Windows „Lite” lub własny build

  • NTLite — stwórz własnego Windowsa bez bloatu
  • Ameliorated Windows — specjalna edycja bez telemetrii
  • Linux + VM z Windows — dla prawdziwych wojowników wolności

🔚 Podsumowanie

  • ✔️ Tak – możesz z Windowsa 11 zrobić cichą, wydajną, nieszpiegowską stację offline
  • ⚠️ Ale wymaga to ręcznych działań i pełnej kontroli nad siecią
  • 🖥️ Adobe działa offline, o ile zadbasz o aktywację i izolację

Zadbaj o backup systemu po oczyszczeniu. Czysty system = spokój na lata.
